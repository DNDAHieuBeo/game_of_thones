# 🛡️ Chronicles of Ice and Fire

A premium, data-driven **Game of Thrones Character Search and Lore Archive** built with modern web technologies. Explore the vast world of Westeros and beyond, featuring real-time data from the ThronesAPI.

![Chronicles of Ice and Fire - Preview]

## 🏛️ Features

- **Live Character Archive**: Fetching data for over 50 characters directly from the `ThronesAPI`.
- **Lore Enrichment Engine**: Every character's profile is enhanced with deterministic lore, including RPG-style statistics (Combat, Intelligence, Diplomacy, Command) and immersive biographies.
- **Dynamic Realm Exploration**: Integrated with the Continents API to explore the various regions of the world (Westeros, Essos, etc.).
- **Smart Filtering & Search**: Instant, persistent filtering by House (Stark, Lannister, Targaryen, etc.) and real-time search by name or title.
- **Cinematic Detail View**: Expanded modal dialogs with rich typography, parchment textures, and detailed character portraits.
- **Responsive & Immersive UI**: A custom-themed design system using Crimson, Gold, and Parchment tones, complete with "aged paper" textures and smooth GoT-themed animations.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand) (Global UI State)
- **Data Fetching**: [TanStack Query v5](https://tanstack.com/query/latest) (Caching & API synchronization)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [tw-animate-css](https://www.npmjs.com/package/tw-animate-css)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.x or later)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/game-of-thrones.git
   cd game-of-thrones
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Navigate to `http://localhost:5173`** in your browser to begin your journey through the Citadel archives.

## 📜 Archival Data Source

All character and continent data is sourced from the excellent [ThronesAPI](https://thronesapi.com/).
